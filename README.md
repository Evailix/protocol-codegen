# Генератори

## Writer
Генерує код, який пакує значення полів у масив байтів відповідно до бітової схеми.

### Приклад використання

```
file_path = Path(r"file.pickle") # шлях до файла
function_name = "write" # назва функції на виході
```


## Reader
Генерує код, який відновлює значення полів із масиву байтів.

### Приклад використання

```
file_path = Path(r"file.pickle") # шлях до файла
function_name = "read" #назва вихідної фукнції
padding = 0 # відсутп в байтах для пропуску непотрібних даних
```
